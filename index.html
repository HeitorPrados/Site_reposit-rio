Aprendemos a utilizar o figma como uma ferramenta pré-código. Resumindo, realizamos um projeto nele de como seria nossa versão final do site já com o HTML e CSS. Portanto, isso facilita o processo de criação do site, pois realizando esse percurso já teríamos uma imagem visual de como criar o sistema.

CSS

Aprendemos a utilizar o CSS como uma ferramenta de estilização. Resumindo, o CSS nos permite definir como os elementos HTML do nosso site serão apresentados ao usuário. Portanto, com o CSS, podemos criar divs, adicionar cores, margens, espaçamentos e alinhamentos. Isso facilita o processo de criação de um site. Nós vimos por último como usar o media para deixar o site adaptável para diferentes dispositivos e tamanhos de tela. Particularmente, o CSS é minha linguagem front-end favorita.

HTML

O HTML é a linguagem base para a criação de sites, que deu origem ao CSS e JavaScript. Com ele, estruturamos o conteúdo, definindo textos, imagens e links. Assim, o HTML facilita a criação do site, fornecendo a base para aplicar estilos e funcionalidades. Ele ajuda a manter o conteúdo organizado e acessível.

Trello

O Trello é uma ferramenta de gerenciamento de projetos que organiza tarefas visualmente usando quadros, listas e cartões. Ele permite adicionar descrições, datas, etiquetas e anexos às tarefas, facilitando a colaboração e o acompanhamento do progresso. Ideal para aumentar a produtividade em projetos pessoais e profissionais. Ajuda bastante na produtividade e colaboração quando se está em grupo, simples e prático, um ótimo gestor de tarefas.

Imagens da Plataforma

Python

Python é uma linguagem de programação versátil e fácil de aprender, usada em desenvolvimento web, automação, análise de dados e IA. Suas bibliotecas facilitam o desenvolvimento eficiente, sendo popular entre iniciantes e profissionais. Gosto bastante, no back-end é a linguagem que eu mais tenho afinidade.

Redes

No Cisco vimos sobre redes de computadores, onde na maior parte das aulas criávamos sistemas de conexão entre eles e outros dispositivos, como switches e roteadores. O processo para checar a funcionalidade da rede se baseava em testar o envio de uma mensagem para um computador e observar se a mensagem chegava de volta ou não. Na minha opinião foi um conteúdo bem simples, mas até que interessante de aprender.
